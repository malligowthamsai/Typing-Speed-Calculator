# ⌨️ Typing Speed Tester (Java Swing)

A simple desktop application to test your typing speed, accuracy, and performance using Java Swing.

## 🚀 Features

- Random sentence generation
- Real-time typing input
- Calculates:
  - ⏱ Time taken
  - ⚡ Words Per Minute (WPM)
  - 🎯 Accuracy
- Clean and user-friendly GUI

## 🖥 How It Works

1. Click **Start**
2. A random sentence will appear
3. Type the sentence in the input area
4. Click **Finish**
5. View your results:
   - Time taken
   - Typing speed (WPM)
   - Accuracy %

## ▶️ How to Run

```bash
javac TypingTester.java
java TypingTester
